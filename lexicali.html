Lexicali

A Ruby application for finding anagrams, prefixed words, and creating word ladders

Goal:

I created a 3 part application for finding anagrams and prefixed words, and creating word ladders given a start and end word. The returned lists are valid words of a standard lexicon.


Languages Used:

Ruby, JQuery, JavaScript, XML & CSS
